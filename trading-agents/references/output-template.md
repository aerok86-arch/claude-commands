# 최종 리포트 템플릿

5단계 워크플로우가 끝나면 아래 형식으로 최종 응답을 구성한다. 굵게 표시된 항목은 반드시 채운다.

```markdown
> ⚠️ 투자 조언이 아닌 시뮬레이션된 리서치 토론입니다. 실제 투자 판단은 본인 책임하에 추가 검증 후 내리세요.

# [종목명]([티커/코드]) — TradingAgents 분석 (YYYY-MM-DD)

**최종 판단**: BUY / HOLD / SELL
**확신도**: 높음 / 보통 / 낮음
**핵심 논지**: (한 줄)

## 1. 애널리스트 팀 요약
- **Fundamentals**: ...
- **Sentiment**: ...
- **News**: ...
- **Technical**: ...

## 2. 리서치 토론 (Bull vs Bear)
- **Bull**: ...
- **Bear**: ...
- **Research Manager 판정**: 현재 [Bull/Bear] 우위. 반전 조건: ...

## 3. 트레이더 계획
- 방향: ... / 목표가: ... / 진입구간: ... / 손절가: ... / 투자기간: ...

## 4. 리스크 관리 토론
- **Risky**: ... / **Neutral**: ... / **Safe**: ...
- **Risk Manager 조정안**: ...

## 5. 펀드매니저 최종 판단
- **판단**: BUY/HOLD/SELL, 확신도 [높음/보통/낮음]
- **근거 요약**: (3줄 이내)
- **주요 리스크**: ...
- **무효화 조건** (이 판단을 재검토해야 하는 트리거): ...

## 출처
- (공시/기사/영상 링크 목록)
```

## 작성 원칙
- 확신도는 근거 데이터의 일관성(4개 애널리스트가 같은 방향을 가리키는지)에 따라 정한다 — 갈릴수록 낮음.
- [미확인] 항목이 많으면 확신도를 자동으로 낮춘다.
- 숫자는 가능한 한 구체적으로(예: "매출 YoY +12%", "목표주가 대비 상승여력 18%") 쓰고, 정성적 표현만 남발하지 않는다.
